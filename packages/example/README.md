## 示例小程序

基于 Taro + TypeScript + CSS Modules 构建小程序


## 前置条件

- 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 安装 Taro cli: `yarn global add @tarojs/cli`


## 项目开发

1、安装依赖

```bash
npm install
```

2、启动项目

```bash
npm run dev:weapp
```

3、打开「微信开发者工具」导入项目，预览体验。
